### Clone the project

```shell
git clone git@github.com:dany338/backend-shopping-cart.git
```

### Setup

Using npm:

```shell
npm install
```

### Local Development

```shell
npm run dev
```

### Production
```shell
npm start
```

### Endpoints

#### `Add a product to cart:`
- http://localhost:3001/api/cart
```json
// method: POST:
{
    "productId": 1,
    "quantity": 2
}

// Response:
[
    {
        "product": {
            "id": 1,
            "name": "Product 1",
            "price": 10,
            "image": "/images/products/1623735-00-A_0_2000.jpg",
            "rating": 1
        },
        "quantity": 2
    }
]
```

#### `Modify the quantity of a product in the cart:`
- http://localhost:3001/api/cart/1
```json
// method: PUT:
{
    "quantity": 5
}

// Response:
[
    {
        "product": {
            "id": 1,
            "name": "Product 1",
            "price": 10,
            "image": "/images/products/1623735-00-A_0_2000.jpg",
            "rating": 1
        },
        "quantity": 5
    }
]
```

#### `Delete a product from the cart:`
- http://localhost:3001/api/cart/1
```json
// method: DELETE:

// Response:
[]
```
